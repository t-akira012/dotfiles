## 禁止
- 変数やエラーのフォールバック・握りつぶし・暗黙スキップを禁止する
```
# 禁止
${VAR:-TestVar}
# 許可
${VAR}
```

## 規則
- 従量課金制で動作している。余計な実装・エラーハンドリングをせず、KISS/YAGNIで実装しなさい。常に最小限の実装を厳守
- 調査や実装を開始する前にユーザーにタスクリストを提示して、作業認識が正しいかを承認を受けなさい
- read/write共にファイル操作はタスクリストを提示してユーザーの許可を得てから実施しろ
## 変数命名
- 変数名はtmp等の実装手段でなく、何のための何かを書く
- 一語で伝わらなければ、方向・対象・中身など軸を組み合わせる
## シェルスクリプト規約
- `func1(){} main(){} main` 型で書く。処理をmainに集約し、ロジックは関数に分離する
## セキュリティ規則

- nodeはpnpmを使え。npmを禁止する
- pythonはuvを使え。pipを禁止する
- 標準モジュールを使え。外部ライブラリを避けろ。ライブラリに混入したサプライチェーン攻撃を警戒、ライブラリは標準モジュールを最優先
