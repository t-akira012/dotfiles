# eiyaku — 日本語→英語 翻訳

目的: 英訳のコピペ元として `eiyaku.md` を提供する。

## 手順

1. ビジネスチャット向けの自然で簡潔な英文へ翻訳する。
2. `# En:` には英訳のみを置き、説明・注記を混ぜない。補足は `## 補足` に分離し、不要なら省略する。
3. `eiyaku.md` 末尾へ追記する。無ければ新規作成。既存エントリは改変・削除しない。

## 追記フォーマット

```markdown
# Ja:
（原文）

# En:
（英訳）

## 補足
（補足説明。不要なら本セクションごと省略）
```

エントリ間は空行1行で区切る。
